# Component Date

This file provides samples of parsed results for components extending the `component_date` class and using the `search_component_date` trait.
The results are generated by the `resolve_query_object_sql` method.
Use component_tipo `test145` for samples.

### 1. Operator !* (Is Empty)
Param $query_object:
```json
{
    "q": "only_operator",
    "q_operator": "!*",
    "path": [
        {
            "name": "date",
            "model": "component_date",
            "section_tipo": "test3",
            "component_tipo": "test145"
        }
    ],
    "table_alias": "te3",
    "table": "matrix_test"
}
```
Parsed result:
```json
{
    "q": "only_operator",
    "q_operator": "!*",
    "path": [
        {
            "name": "date",
            "model": "component_date",
            "section_tipo": "test3",
            "component_tipo": "test145"
        }
    ],
    "table_alias": "te3",
    "table": "matrix_test",
    "data_path": ["components", "test145", "dato", "lg-nolan"],
    "type": "jsonb",
    "sentence": "NOT (te3.date @? '$.test145[*]') "
}
```
SQL resolved:
```sql

(te3.section_tipo = 'test3') 
AND NOT (te3.date @? '$.test145[*]')
```
