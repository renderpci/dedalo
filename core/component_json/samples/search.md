# Component JSON

This file provides samples of parsed results for components extending the `component_json` class and using the `search_component_json` trait.
The results are generated by the `resolve_query_object_sql` method.
Use component_tipo `test18` for samples.

### 1. Operator !* (Is Empty)
Param $query_object:
```json
{
    "q": "only_operator",
    "q_operator": "!*",
    "path": [
        {
            "name": "json",
            "model": "component_json",
            "section_tipo": "test3",
            "component_tipo": "test18"
        }
    ],
    "table_alias": "te3",
    "table": "matrix_test"
}
```
Parsed result:
```json
{
    "q": "only_operator",
    "q_operator": "!*",
    "path": [
        {
            "name": "json",
            "model": "component_json",
            "section_tipo": "test3",
            "component_tipo": "test18"
        }
    ],
    "table_alias": "te3",
    "table": "matrix_test",
    "type": "string",
    "sentence": "(te3.misc IS NULL\nOR NOT EXISTS (\n SELECT 1\n FROM jsonb_array_elements(te3.misc->'test18') AS elem\n WHERE elem->>'value' IS NOT NULL\n AND elem->>'value' != ''\n )\n)"
}
```
SQL resolved:
```sql
SELECT * FROM matrix_test AS te3 WHERE 
(te3.section_tipo = 'test3') 
AND ((te3.misc IS NULL
OR NOT EXISTS (
 SELECT 1
 FROM jsonb_array_elements(te3.misc->'test18') AS elem
 WHERE elem->>'value' IS NOT NULL
 AND elem->>'value' != ''
 )
))
```
