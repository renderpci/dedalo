# component_relation_index

This file provides samples of parsed results for components extending the `component_relation_index` class and using the `search_component_relation_index` trait.
The results are generated by the `resolve_query_object_sql` method.
Use component_tipo `test25` for samples.

### 1. Operator !* (Is Empty - Not Indexed)
Param $query_object:
```json
{
    "q": "only_operator",
    "q_operator": "!*",
    "path": [
        {
            "name": "relation_index",
            "model": "component_relation_index",
            "section_tipo": "test3",
            "component_tipo": "test25"
        }
    ],
    "table_alias": "te3",
    "table": "matrix_test"
}
```
Parsed result (Example with some references found):
```json
{
    "q": "only_operator",
    "q_operator": "!*",
    "path": [
        {
            "name": "relation_index",
            "model": "component_relation_index",
            "section_tipo": "test3",
            "component_tipo": "test25"
        }
    ],
    "table_alias": "te3",
    "table": "matrix_test",
    "sentence": "te3.section_id NOT IN (1,2,3)"
}
```
SQL resolved:
```sql
SELECT * FROM matrix_test AS te3 WHERE te3.section_id NOT IN (1,2,3)
```

```
Parsed result (Example with some references found):

    "q": "only_operator",

    "path": [
        {
            "name": "relation_index",
            "model": "component_relation_index",

        }

    "table_alias": "te3",

```

### 3. No references found (Empty result case)
If `get_references_to_section` returns an empty array:
Operator `*` results in `sentence: "1=0"`.
Operator `!*` results in `sentence: "1=1"`.

SQL resolved (Not Empty):
```sql
SELECT * FROM matrix_test AS te3 WHERE 1=0
```

SQL resolved (Empty):
```sql
SELECT * FROM matrix_test AS te3 WHERE 1=1
```
